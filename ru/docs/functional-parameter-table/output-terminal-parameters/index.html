      5.6 Группа Р5 Параметры вывода терминала

      ℹ️
      Описание атрибутов параметров инвертора
      • ✅: Указывает на то, что значение этого параметра можно изменить, когда инвертор находится как в остановленном, так и в рабочем состоянии.
      • ✳️: Указывает на то, что значение этого параметра нельзя изменить, когда инвертор находится в состоянии работы.
      • ❎: Указывает на то, что значение этого параметра является фактическим измеренным значением и не может быть изменено.
